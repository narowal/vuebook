VueJs Tutorial

How to install it?
1- clone the repo using command in your www directory: git clone https://github.com/tayyabkhanrind/vuebook.git
2- install dependencies using npm command: npm update
3- browse the application in your browser and just enjoy it.


What vueBook included?
1- VueBook included basic usage of VueJs
2- VueJs Components
3- Show/Hide Component
4- Usage of modal in VueJs
5- Calling restfull API using VueJs
6- Data binding using VueJs
7- String Limiter
